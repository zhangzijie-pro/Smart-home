# 智能家居
- 2024-2025一带一路暨金砖国家技能发展与技术创新大赛 (国二作品)

题目要求:

    需要使用手机，语音和按键来进行家居的控制

### 主控使用stm32f407系列
- 其中涉及到: 继电器, LED, 红外人体感应,雨滴传感器, 火焰传感器, 3个舵机,温湿度, 串口触摸屏, SU-03, 烟雾和一氧化碳检测, esp32cam,led变色灯...

# 文件分布
| 目录名                 | 描述                     |
|-----------------------|--------------------------|
| esp32cam/| esp32摄像头与手机app内容 |
| usart_hmi/|串口屏幕显示源码以及配置内容|
| USER/ | main函数入口|
| HARDWARE/| 外设函数|
